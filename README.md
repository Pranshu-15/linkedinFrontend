## 🚀 Quick Start (Frontend)

1. Make sure you have Node.js installed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your `.env` file based on the root documentation (e.g. `VITE_API_URL`).
4. Start the development server:
   ```bash
   npm run dev
   ```

This frontend is built with **React 19**, **Vite**, **Tailwind CSS**, **Framer Motion**, and **Socket.io-client**.
